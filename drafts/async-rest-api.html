Looking for some more challenges?

Review the official tutorial. It's long but well worth a read. Implement async background tasks, database migrations, and auth. Abstract out the application configuration to a separate file. In a production environment, you'll probably want to stand up Gunicorn and let it manage Uvicorn. Review Running with Gunicorn and the Deployment guide for more info. Check out the official uvicorn-gunicorn-fastapi Docker image as well.
