Git Tips

Reset

Reset current git repo to match remote

# Save work
git co -b save-work
git co master

git fetch origin
git reset --hard origin/master

Remove wrong commit from master. Reset reassign the branch pointer

HEAD   is the commit currently siting on
^HEAD  this commit parent
^^HEAD this commit grandparent

$ git co master
$ git reset --hard ^HEAD

Reset last commit

git reset --hard HEAD~1

Contribute

  1. Make Fork on webpage
  2. Create local clone
  3. Add remote upstrem to original repo
$ git remote add upstream https://github.com/octocat/Spoon-Knife.git
  1. View remote
$ git remote -v
  1. Sync a fork
$ git fetch upstream
  1. Branch set to track upsream master
$ git branch --set-upstream-to=upstream/master master
  1. Create new brnch for working on
$ git checkout -b <branch-name>
  1. Create changes, commit and push to origin remote
$ git push -u origin <branch-name>
