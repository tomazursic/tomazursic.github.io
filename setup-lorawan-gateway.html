Kersing version

Warning

This pktfwd is substituted with Semtech version of pktfwd

STEP 1:

Make copy of example-config.json and set MQTT

STEP 2:

Edit mppktfwd_config/local_conf.json and set server block

STEP 3:

Run setup script

cd ttn-gw/ && ./setup build && ./setup install
